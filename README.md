## 💻 Sobre o projeto

Este é um projeto de uma API REST, para uma aplicação de gerenciamento recursos financeiros de clubes.

Possui as funcionalidades de:

- Ver todos os clubes;

- Cadastrar clubes;

- Permitir clubes a fazer consumos com seu valor disponível;
